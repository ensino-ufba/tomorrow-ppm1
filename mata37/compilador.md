# Compilador

Compilador é um programa que lê um código-fonte em uma linguagem de programação e gera um arquivo executável correspondente.

---

# Alguns compiladores de C++

- GNU C Compiler (gcc)
- Clang / LLVM
- Visual C++ Compiler

---

# Compilador gcc

```bash
g++ alo.cc
./a.out
```

---

# Compilador gcc

```bash
g++ alo.cc -o alo
./alo
```

---

# Compilador gcc

```bash
g++ -Wall alo.cc -o alo
./alo
```
